<h2>Description</h2>
This is the starter code used in Boot.dev's [Learn Pub/Sub](https://learn.boot.dev/learn-pub-sub) course.

<h2>Motivation</h2>
None, this is just to satisfy the test.

<h2>Quick Start</h2>
./rabbit.sh start
go run ./cmd/server
go run ./cmd/client

<h2>Usage</h2>
After launching RabbitMQ and the server, launch at least two clients in new Ubuntu instances.
Provide a different username for each instance player.
Each player needs to spawn at least one army.
Move the armies until they are in same territory and fight.
